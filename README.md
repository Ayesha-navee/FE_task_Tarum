# PYME Landing Page

A modern, responsive landing page for PYME - AI Co-Founder for Business, built with Next.js and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: React

## Features

✨ Fully responsive design (mobile, tablet, desktop)
🎨 Modern gradient design system
⚡ Fast performance with Next.js optimization
🔧 Reusable component architecture
📱 Mobile-first responsive approach
♿ Accessible markup

## Project Structure

```
components/         # Reusable React components
app/               # Next.js app directory
├── layout.tsx     # Root layout
├── page.tsx       # Main page
└── globals.css    # Global styles
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Sections

1. **Header** - Responsive navigation
2. **Hero** - Main landing section with dashboard preview
3. **Department Intro** - Business overview
4. **Product** - Product features
5. **Operations** - Workflow automation
6. **Team** - Team management
7. **Footer** - Links and info

## Responsive Design

- Mobile: Optimized layouts for small screens
- Tablet: Adapted for medium screens
- Desktop: Full-width design for large screens

## Customization

- **Colors**: Edit `tailwind.config.js`
- **Content**: Update individual components
- **Spacing**: Modify Tailwind theme values
